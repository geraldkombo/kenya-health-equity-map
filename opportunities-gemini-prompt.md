# Gemini Prompt: Find Submission Opportunities for the Kenya Health Equity Map

## Project Summary
The **Kenya Health Equity Map** is a fully deployed, open-data platform for community-led health advocacy. It visualizes infrastructure disparities across all 47 Kenyan counties using a composite **Priority Gap Score (PGS)** derived from poverty rates, travel time to facilities, facility density, and population pressure. Built with Next.js 15, Tailwind CSS, CARTO map tiles, and OpenStreetMap data.

- **Live URL:** https://geraldkombo.github.io/kenya-health-equity-map/
- **Source:** https://github.com/geraldkombo/kenya-health-equity-map
- **Status:** Production-ready, live, open-source (no tracking, no login, no cookies)
- **Abstract:** Already submitted to Amref CSS Forum 2026
